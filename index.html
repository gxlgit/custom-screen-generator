Custom Screen Generator

About

Build a custom screen! Use cases include:

  • Build official custom screens for @work partners in Foundry.
  • Build demos for prospects.
  • And more!

Specs

  • Version 1.1
  • Best for screens greater than 1200pxc wide.

Instructions

  1. Pick an available template to get started.
  2. Explore the configurable elements and their customizable options.
  3. Under the "Review" tab, click the "copy code" button. The custom screen will be copied to your clipboard.
  4. Enter Foundry and paste the code into the custom content editor. Tweak as desired.
  5. You will LOSE PROGRESS if you switch templates or reload the page.
