Custom Screen Generator

About

Build a custom screen! Use cases include:

  • Build official custom screens for @work partners in Foundry.
  • Build demos for prospects.
  • And more!

Specs

  • Version 1.1
  • Best used for browser widths greater than 1200 pixels.
  • Requires Javascript. Built on React framework.
  • For internal use ONLY

Instructions

  1. Pick an available template to get started.
  2. Explore the configurable elements and their customizable options.
  3. Choose colors by entering 6 digit #hex codes or by using the color wheel.
  4. Make sure the button URL is using 'https://' syntax.
  5. When finished find the "Review" tab and click the "copy code" button. The custom screen will be copied to your clipboard.
  6. Enter Foundry and paste the code into the custom content editor. Tweak copy and formatting if desired.
  7. You will LOSE PROGRESS if you switch templates or reload the page.

FAQS

How do I add list-items?

  • Paste code into Foundry custom content editor. Highlight the text you want listed. Hit the unordered or ordered list button.

How do I add multiple paragraphs?

  • Write all copy into the apps text editor. Copy and paste code into Foundry custom screen editor. Place cursor where you want to break the paragraph and hit Enter.

Can I use custom fonts?

  • Custom fonts cannot be added yet. Please contact a developer if you need to add custom fonts.

Can I add images?

  • Images cannot be added yet. Please contact a developer if you need to add images.
